Under the hood

Three containerized services deployed via Bicep infrastructure-as-code, with CI/CD through GitHub Actions. Cloud-agnostic by design — the whole stack runs anywhere Docker runs.

siege-api

FastAPI · SQLAlchemy async · Playwright for image generation

siege-frontend

React 18 · Vite · TypeScript · Tailwind · shadcn/ui

siege-bot

discord.py client + FastAPI HTTP sidecar

PostgreSQL · Key Vault · Application Insights · Log Analytics

Run it for your own clan

RSL Siege Manager is fully open source and deliberately cloud-agnostic. Three paths — pick the one that fits your setup.

👀

Try it locally

~5 minutes

Docker Compose + auth disabled + seed data. Click around the UI with zero Discord setup.

🏠

Self-host anywhere

Recommended

~30 minutes

VPS or home server + Docker. Works on any box that runs containers. No cloud bill.

Deploy to Azure

~1–2 hours
